# Шаблон Neo v3 (Web)

Добро пожаловать в репозиторий веб-шаблона **Neo v3** (текущая версия в архиве: **3.7**). Данный шаблон предназначен для создания сайта игрового проекта на основе LRWEB.
(CS2, возможная адаптация под CSGO) 

## 📋 Системные требования

> 🔥 **КРИТИЧЕСКИ ВАЖНО:** Для корректной работы шаблона на веб-хостинге **ОБЯЗАТЕЛЬНО** должна быть выставлена версия **PHP 7.4**. На других версиях PHP работа не гарантируется.

## 🚀 Быстрый старт

Для установки шаблона выполните следующие шаги (подробности см. в инструкции):

1.  Убедитесь, что версия PHP на хостинге — **7.4**.
2.  Загрузите файлы из архива `neo 3.7` в корень вашего сайта.
3.  Настройте права доступа (CHMOD) для необходимых папок и файлов (см. `instructionsetup.md`).
4.  Перейдите на свой сайт и следуйте инструкциям веб-установщика.

## 📖 Документация

Подробная пошаговая инструкция по установке и настройке находится в файле:
📄 **[instructionsetup.md](instructionsetup.md)**

* Для работы потребуется **Steam Web API ключ**.
* Если вы используете систему **IKS ADMIN v3**, обязательно ознакомьтесь со специальным разделом в инструкции по установке касательно изменений в `DB.php` и базе данных.
