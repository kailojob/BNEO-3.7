# Подробная инструкция по установке шаблона Neo v3

Здесь вы можете прочитать подробную инструкцию по установке шаблона Neo v3 (версия архива 3.7). Пожалуйста, следуйте всем пунктам внимательно.

## Этап 1: Подготовка к установке

Перед началом убедитесь, что ваш хостинг соответствует требованиям, и подготовьте файлы.

### 1.1 Настройка PHP
> ❗ **ОБЯЗАТЕЛЬНОЕ ТРЕБОВАНИЕ:** Выставите версию **PHP 7.4** в панели управления вашим веб-хостингом. Шаблон разработан именно под эту версию.

### 1.2 Загрузка файлов
Залейте файлы из архива **"neo 3.7"** в корневую директорию вашего сайта.

---

## Этап 2: Установка прав доступа (CHMOD)

Для корректной работы системы необходимо установить права **777** (полный доступ) на следующие директории и файлы.

### 2.1 Основные директории
Установите права **777** на всю папку `storage/` **рекурсивно** (на саму папку, все подпапки и файлы внутри неё).

### 2.2 Модули
Установите права **777** на следующие папки модулей:
* `module_block_main_servers`
* `module_block_main_statistics`
* `module_page_adminpanel`
* `module_page_faq`

### 2.3 Темы и конфигурация
Установите права **777** на папку и файл:
* Папка: `app/templates/neo_remastered/themes/custom`
* Файл: `app/templates/neo_remastered/colors.json`

---

## Этап 3: Веб-установка

После завершения настройки прав перейдите на свой сайт. Автоматически начнется процесс установки шаблона и баз данных. Следуйте шагам установщика:

1.  **Выбор языка:** Выберите язык интерфейса вашего сайта.
2.  **Настройка базы данных:** Укажите данные подключения к базе данных (хост, пользователь, пароль, имя БД), в которой вы хотите хранить таблицы шаблона. Если БД уже существует, введите её данные.
3.  **Steam API ключ:** Укажите ваш **STEAM WEB API ключ**. Если у вас его нет, создайте его здесь: [https://steamcommunity.com/dev/apikey](https://steamcommunity.com/dev/apikey).
4.  **Администратор:** Укажите **SteamID** главного администратора сайта.
5.  **SEO настройки:** Введите данные о вашем проекте/сервере. Эти данные будут использоваться для поисковой оптимизации (SEO).

---

## ⚠️ Примечание для пользователей IKS ADMIN v3

Если вы используете систему управления **IksAdmin v3**, вам необходимо выполнить дополнительные действия после установки для интеграции:

1.  **В файле `DB.php`:** найдите и измените `IksAdmnin` на `IksAdminNew`.
2.  **В базе данных:** в таблице `lvl_web_servers` измените значение `IksAdmnin` на `IksAdminNew`.
